# Nurbol Sultanov — Data Analyst Portfolio

Live site: https://everinurmind.github.io/portfolio-setup-and-mobile-first

A personal portfolio website built mobile-first with vanilla HTML, CSS, and JavaScript. Showcases data analytics projects, skills, and a live code section with syntax highlighting.

## Features

- Mobile-first responsive design
- Glassmorphism UI with animated background
- Typewriter effect in hero section
- Project filter by technology (SQL, Python, Power BI)
- Syntax-highlighted code showcase (SQL + Python)
- Contact form with Formspree integration
- Smooth scroll and fade-in animations

## Tech Stack

HTML5 · CSS3 · JavaScript · Prism.js · Google Fonts (Inter, JetBrains Mono)
